# DK Party

### Запуск

1. Установить разрешение экрана 1920 x 1080

2. Установить английскую раскладку клавиатуры.

3. Установить в UC Browser масштаб всех страниц - 80%

4. Отредактировать файл config.ini

5. Заполнить accounts.txt аккаунтами по маске:
```
{login}:{password}:{server1}.{server2}.{serverN}
```
Пример:
```
test@provider.loc:password:77.78.99.100
```

6. Запустить party.exe


### Примечания:

+ файл с аккаунтами может иметь произвольное название, следовательно 
поправить параметр accounts_path в файле конфига
+ рекомендуется указывать количество серверов на одном аккаунте кратное 4

Компиляция под Windows:
```sh
$ pyinstaller --onefile --icon=icon.ico party.py 
```
